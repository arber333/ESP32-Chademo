# CHAdeMOSoftware
This code will help you add CHAdeMO DC fast charging to your EV, whether it is OEM or DIY.  Currently Untested

Based on https://github.com/Isaac96/CHAdeMOSoftware

# Required hardware 
- CHAdeMO connector  
- 2x contactors  
- ESP32-Chademo Board
- Relay  
- 1k pullup resistor   
- Isabellenhutte IVT or IVT-S shunt
